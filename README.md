# 🧠 Stroke Detection System

## 📌 Project Overview
The **Stroke Detection System** is a machine learning-based project developed to predict the likelihood of stroke in patients using medical and lifestyle data. The system combines **Machine Learning (ML), Clustering techniques, and Artificial Neural Networks (ANN)** to analyze patterns in the dataset and improve prediction accuracy.

This project aims to support early detection of stroke risk, which can help in timely medical attention and potentially save lives.

---

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Stroke risk prediction using Machine Learning models
- Clustering to identify hidden patterns in patient data
- Artificial Neural Network (ANN) for deep learning-based prediction
- Model evaluation using accuracy, precision, recall, and F1-score
